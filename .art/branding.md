# OxideDock Brand Tokens

## Core Positioning

- Name: `OxideDock`
- Story: raw steel oxidizing into a protective patina, symbolizing software that starts fast and becomes refined and resilient.
- Personality: sturdy, modern, calm, fast.
- Promise: native-speed desktop development without rough edges.

## Taglines

- Primary: `Raw speed, refined desktop.`
- Alternative: `Built fast. Finished beautifully.`
- Alternative: `From raw metal to ready-to-ship.`
- Alternative: `Performance forged for desktop.`
- Alternative: `Where fast apps come to dock.`

## Product Description

- Short: `OxideDock is a Rust + Vue desktop foundation that turns native performance into polished, ship-ready apps.`
- Extended: `OxideDock gives you a clean Rust + Vue desktop stack focused on speed, reliability, and product-quality UX. Like steel forming patina, it gets stronger and more refined with every iteration.`

## Typography Tokens

- Headline: `Sora` or `Space Grotesk`
- Body: `Inter` or `Manrope`
- Mono: `JetBrains Mono`

## Voice Guidelines

- Tone: clear, confident, practical.
- Principle: fewer buzzwords, more outcomes.
- Preferred language: "fast and reliable" over "magical and disruptive."

## Logo Prompt

```text
Design a clean, modern logo for “OxideDock”. Concept: steel oxidizing into a refined protective patina. Create a geometric symbol combining an “O” ring and a subtle dock/harbor notch. Style: minimal, flat, premium tech, high contrast, no clutter. Palette: deep steel (#1F2933), oxide orange (#C65A3A), mist (#E9EEF2). Include symbol + wordmark lockup. Ensure legibility at small sizes and strong silhouette. White background. No 3D mockups.
```

## App Icon Prompt

```text
Create a 1024x1024 app icon for “OxideDock”. Rounded square background in deep steel (#1F2933). Center a bold minimal oxide ring with a small dock notch, using oxide orange (#C65A3A) accent. Flat vector style, crisp edges, subtle depth only through shape layering (not gradients). Must remain recognizable at 32x32. Premium desktop developer-tool aesthetic.
```

## Favicon Prompt

```text
Generate a simple favicon mark for OxideDock: a monoline “O” with a dock notch, single color version and two-color version (deep steel + oxide). Extremely minimal, pixel-clear at 16x16.
```
